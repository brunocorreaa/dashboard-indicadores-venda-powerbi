# Sales Performance Dashboard - Power BI

Projeto de análise de indicadores de vendas desenvolvido em Power BI com dados anonimizados. O objetivo é demonstrar a construção de um dashboard de performance comercial com foco em indicadores operacionais e gerenciais.

## Sobre o projeto

Este dashboard apresenta uma visão consolidada do desempenho de vendas, permitindo acompanhar indicadores estratégicos do negócio e apoiar análises de performance comercial.

Entre os principais indicadores analisados estão:

- Venda Financeira
- LB% (Margem Bruta)
- Ticket Médio
- Quantidade de Vendas

Além da visão executiva, o projeto também permite análises detalhadas por dia, filial e setor de negócio.

# Indicadores de Venda por Setor

Nesta visão, o foco está na análise comparativa da performance financeira entre diferentes setores do negócio.

O dashboard apresenta:

- Comparação de **Venda Financeira entre Ano Atual (AC) e Ano Anterior (AA)**
- Distribuição da performance por setor
- Identificação de setores com maior crescimento ou variação de vendas

Essa análise ajuda a entender como cada área contribui para o resultado geral da operação.

# Modelo de Dados

O projeto também inclui a modelagem de dados utilizada para estruturar as análises no Power BI.

O modelo foi construído com relacionamentos funcionais entre as tabelas, permitindo a correta agregação das métricas e o cruzamento das dimensões analíticas utilizadas no dashboard.

Essa etapa é fundamental para garantir consistência nos indicadores e permitir análises confiáveis.

# Estrutura do Projeto

O desenvolvimento do projeto envolveu as seguintes etapas:
